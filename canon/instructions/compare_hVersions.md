To compare hVersions, first check the hTag for the current hProject. This can be found in .helmsman/hVersion.md. The contents should be a tag (example v0.0.1) which corresponds to a tag in the helmsman github repo. Check out the tag on git and compare the contents of canon/c_hDocs to /hDocs. c_AGENTS.md, c_cursorignore.md, c_glossary.md, c_HELMSMAN.md, c_master.md, and c_overlay.md MUST match verbatim the documents found in hDocs. If the documents do not match, then drift has occurred.
